El Viewpor es el area visible del usuario de una página web.

La ventana grafica varia segun el dispositivo y sera mas pequeña en un telefono movil que en la pantalla de un ordeandor.

Antesde las tablets y los telefonos moviles, las paginas webs estaban diseñadas solo para pantallas de ordenador, y era comun que las paginas web tuvieran un diseño estatico y un tamaño fijo.

Luego cuando comenzamos a navegar por internet usando tablet y moviles, las paginas web de tamaño fijo eran demasiado grandes para caber en la ventana grafica. Para solucionar esto, los navegadores de esos dispositos redujeron todo la pagina web para que se ajustar a la pantalla.

¡¡Esto no fue perfecto!! Pero si una solucion rapida.
